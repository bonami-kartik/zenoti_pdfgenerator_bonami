
## Quick start

1.  Make sure that you have Node.js and npm installed.
2.  Clone this repo using `git clone https://github.com/Default-in/zenoti_pdf_generator_frontend.git`
3.  Move to the appropriate directory: `cd zenoti_pdf_generator_frontend`.<br />
4.  Run `npm install` in order to install dependencies.<br />
4. _At this point you can run `npm start` to see theexample app at `http://localhost:3000`._
